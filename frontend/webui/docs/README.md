# ComfyEmotionGen 웹 사용 설명서

ComfyEmotionGen은 ComfyUI 기반의 대규모 이미지 배치 생성 · 큐레이션 도구입니다. 한 번의 템플릿 작성으로 수십 ~ 수천 장을 자동 생성하고, 각 조합의 결과물을 검토 · 분류 · 다운로드할 수 있습니다.

이 문서는 웹 프론트엔드의 화면 구성과 각 기능의 사용 방법을 설명합니다.

## 목차

1. [시작하기](./01-시작하기.md) — 화면 구성, 백엔드 연결, 기본 흐름
2. [작업 탭](./02-작업-탭.md) — 템플릿 / 워크플로우 작성, 작업 실행, 큐 매니저, 세션
3. [통계 탭](./03-통계-탭.md) — 처리 현황 · 작업자 · 시간 분포 차트
4. [갤러리 탭](./04-갤러리-탭.md) — 저장된 이미지 검색, 필터, 휴지통, 데이터셋 익스포트
5. [큐레이션 탭](./05-큐레이션-탭.md) — 조합 단위 검토, 토너먼트 · 비교 뷰, 대표 이미지 지정
6. [템플릿 생성기 탭](./06-템플릿-생성기.md) — 사전 정의 템플릿에서 변수만 채워 새 템플릿 만들기
7. [설정 및 단축키](./07-설정-및-단축키.md) — 환경 설정, 웹훅, 키보드 단축키

## 화면 한눈에 보기

상단에는 항상 **헤더**가 있고, 그 안에 6개의 메인 탭이 있습니다.

| 탭 | 용도 |
|---|---|
| 작업 | 템플릿/워크플로우를 만들고 작업을 큐에 넣어 실행합니다 |
| 통계 | 생성한 작업의 통계, 실패/성공률, 작업자 사용량을 봅니다 |
| 갤러리 | 생성된 이미지를 탐색하고 정리합니다 |
| 큐레이션 | 같은 조합의 여러 후보 중 대표 이미지를 고릅니다 |
| 템플릿 생성기 | 기본 템플릿에 변수만 채워 자기만의 템플릿을 만듭니다 |
| 설정 | 백엔드 주소, 워커, 웹훅, 테마 등 환경을 조정합니다 |

상태 표시기, 세션 선택기, 일시중지/취소 같은 전역 컨트롤도 헤더 영역에 모여 있습니다.

## 빠른 시작

1. **설정** 탭에서 백엔드 주소가 올바른지 확인합니다.
2. 헤더의 백엔드 상태등이 초록색인지 확인합니다.
3. **작업** 탭에서 워크플로우를 불러오고(또는 새로 입력), 노드 매핑을 설정합니다.
4. CEG 템플릿을 작성하거나 **템플릿 생성기**에서 변수만 채워 만듭니다.
5. **실행** 버튼을 눌러 큐에 작업을 추가합니다.
6. **갤러리** 또는 **큐레이션** 탭에서 결과를 검토합니다.

각 단계의 자세한 설명은 위 문서 링크를 참고하세요.
