# 5. 큐레이션 탭

큐레이션 탭은 **같은 조합으로 생성한 여러 후보 이미지 중에서 대표 이미지를 고르는** 전용 화면입니다.

갤러리 탭이 "낱장 이미지" 단위 작업이라면, 큐레이션 탭은 "한 조합당 후보 묶음" 단위 작업입니다. 배치 수를 3~5로 두고 같은 조합을 여러 번 돌린 뒤, 가장 잘 나온 한 장만 골라 데이터셋에 남기는 흐름에 최적화되어 있습니다.

## 5.1 화면 구성

- **상단 도구막대** — 축(그룹 기준) 선택, 뷰 모드, 필터, 썸네일 크기, 설정, 내보내기 등이 있습니다.
- **좌측 사이드바** — 조합 그룹 목록과 미배정 항목 패널이 있습니다. (데스크톱: 상세 뷰 진입 시 표시, 모바일: 시트(Sheet)로 슬라이드)
- **본문(갤러리 / 테이블 / 그리드 / 비교 / 토너먼트)** — 선택한 뷰 모드의 내용이 표시됩니다.
- **필터 바** — 도구막대 아래 접이식으로 상태 필터와 통합 검색바가 있습니다.
- **선택 모드 바** — 선택 모드가 활성화되면 도구막대 아래 선택 개수, 일괄 재생성/다운로드 버튼이 표시됩니다.

## 5.2 축(그룹 기준) 선택

도구막대의 **축** 셀렉트는 "후보를 어떻게 묶을지"를 결정합니다.

### 템플릿 기반 분류

- **현재 편집 중인 템플릿** — 작업 탭에서 작성한 CEG 템플릿의 모든 축을 펼쳐 조합 단위로 묶습니다. 가장 일반적인 선택입니다.
- **저장된 템플릿** — 과거에 저장한 다른 템플릿을 기준으로 묶을 수도 있습니다.

### 자유 그룹화

템플릿 없이 파일명, 메타데이터 키 등 단순 기준으로 묶습니다. 네 가지 모드가 있습니다.

| 모드 | 설명 |
|---|---|
| **파일명별** | 원본 파일명이 같은 이미지를 한 그룹으로 묽니다. |
| **파일명 패턴 파싱** | 파일명을 `_`로 분할하고 `key-value` 형식을 파싱해 메타데이터를 추출합니다. (예: `style-anime_seed-1234` → `style: anime`, `seed: 1234`) |
| **태그별** | 이미지에 붙은 태그를 기준으로 묶습니다. 태그가 없는 이미지는 `(태그 없음)` 그룹으로 분류됩니다. |
| **저장된 템플릿별** | 이미지에 기록된 `cegTemplate` 값을 해시화하여 같은 템플릿을 사용한 이미지를 묶습니다. 템플릿 정보가 없는 이미지는 `(템플릿 없음)` 그룹으로 분류됩니다. |

축을 바꾸면 그룹 목록이 다시 계산됩니다. 계산이 끝나면 사이드바에 그룹이 채워집니다.

## 5.3 갤러리 뷰 (목록)

기본 뷰입니다. 조합 그룹의 대표 후보(승인된 이미지 또는 최신 이미지)를 썸네일 격자로 보여 줍니다.

- 각 카드에는 승인 여부(초록 체크), 이미지 수 배지가 표시됩니다.
- 마우스를 올리면 **호버 카드**가 열려 해당 그룹의 모든 후보 이미지(최대 12장)와 상태 배지(통과/대기/탈락/휴지통), 최근 프롬프트, 메모, 태그 등을 미리 볼 수 있습니다.
- 그룹을 클릭하면 상세 뷰(그리드)가 열려 그 그룹의 모든 후보를 펼쳐 봅니다.
- 이미 대표가 지정된 그룹은 시각적으로 강조됩니다.

### 테이블 뷰

목록 모드에서 **테이블 보기**를 선택하면 조합 목록을 표 형태로 볼 수 있습니다.

| 열 | 내용 |
|---|---|
| 상태 | 대표 이미지 승인 여부 (체크/빈 원) |
| 파일명 | 조합 파일명 (호버 시 미리보기) |
| 메타데이터 | 조합의 메타 태그 |
| 수 | 후보 이미지 개수 |

테이블 뷰는 조합이 많을 때 파일명과 메타데이터를 빠르게 비교할 때 유용합니다.

## 5.4 조합 상세 뷰

그리드의 그룹을 클릭하면 열리는 화면입니다. 한 조합의 모든 후보를 큰 화면에 줄지어 보여 줍니다.

### 상세 헤더 컨트롤

상단에는 다음과 같은 버튼과 정보가 표시됩니다.

- **뒤로가기** — 목록으로 복귀 (`Esc`)
- **이전/다음 조합** — 조합 간 이동 (`K` / `J`)
- **파일명 및 메타 태그** — 현재 조합의 정보
- **설정 드롭다운** — 다음 세 가지 일괄 동작:
  - **모두 리젝** — 승인/탈락 상태가 아닌 모든 후보를 탈락 처리
  - **리젝 취소** — 탈락된 후보를 다시 대기 상태로 변경
  - **선택 취소** — 승인된 대표를 취소하고 모든 상태를 대기 상태로 변경
- **재생성 버튼** — 현재 조합을 다시 생성 (`R`)

### 후보 이미지 상호작용

각 후보 이미지 위에는 인덱스 번호(1~9)가 표시됩니다. 번호 키로 한 번에 대표를 지정할 수 있습니다.

| 키 | 동작 |
|---|---|
| `1` ~ `9` | 해당 번호의 후보를 **대표(승인)** 로 지정 |
| `←` / `h` | 왼쪽 이미지로 포커스 이동 |
| `→` / `l` | 오른쪽 이미지로 포커스 이동 |
| `Enter` / `Space` | 현재 포커스된 이미지를 대표로 선택 |
| `↓` / `j` | 다음 조합으로 이동 |
| `↑` / `k` | 이전 조합으로 이동 |
| `r` / `R` | 이 조합을 다시 생성(시드만 다른 새 후보 추가) |
| `Esc` | 상세 뷰를 닫고 그리드로 복귀 |

이미지를 클릭하면 대표로 선택됩니다. 이미 선택된 이미지를 클릭하면 선택이 취소됩니다.

후보 위에서 우클릭하면 다음 메뉴가 뜹니다.

- **이미지 보기** — 전체 화면 미리보기
- **선택** — 대표로 승인
- **리젝** — 탈락 처리
- **선택 취소** — 승인 상태 해제
- **리젝 취소** — 탈락 상태 해제

### 비교 핀

후보 이미지 위에 비교 버튼(열 아이콘)을 누르면 비교 뷰에 추가됩니다. 2장 이상 핀하면 비교 뷰를 활성화할 수 있습니다. 비교 뷰에서는 **마그나ifier(확대경)** 기능이 활성화되어 마우스를 올리면 해당 영역이 250% 확대됩니다.

### 자동 진행

도구막대 설정의 **자동 다음 이동** 토글을 켜면 대표를 지정하자마자 다음 조합으로 자동 이동합니다. 이때 이미 대표가 지정된 조합은 건너뛰고, **아직 대표가 지정되지 않은 조합**으로 이동합니다. 빠르게 훑어 내려갈 때 매우 편리합니다.

### 자동 탈락 적용

`autoApplyReject` 설정이 켜져 있으면(기본값: true) 한 후보를 승인할 때 나머지 후보는 자동으로 탈락 처리됩니다. 데이터셋에 대표만 남기고 싶을 때 켜 두세요.

### 리젝 숨기기

설정에서 **리젝 숨기기**를 켜면 상세 뷰에서 탈락된 이미지가 숨겨집니다. 대표 선정 후 남은 후보만 보고 싶을 때 유용합니다.

## 5.5 비교 뷰

도구막대의 그리드 서브 모드를 **비교**로 바꾸면 핀해 둔 이미지들을 나란히 봅니다.

- 그리드 뷰에서 이미지 위의 비교 버튼(열 아이콘) 또는 우클릭 메뉴로 핀합니다.
- 핀된 이미지가 2장 이상이어야 활성화됩니다.
- 비교 뷰에서 **마그나ifier(확대경)** 기능으로 이미지 세부 사항을 확대 확인할 수 있습니다.
- 가장 마음에 드는 한 장을 골라 대표로 승격시킬 수 있습니다.

## 5.6 토너먼트 뷰

후보가 많아 한 번에 보기 어려울 때 사용합니다. "이상형 월드컵" 방식으로 두 장씩 짝지어 보여 주고 더 좋은 쪽을 고르면, 이긴 후보들끼리 다시 짝지어 마지막 한 장이 남을 때까지 반복합니다.

### 토너먼트 조작법

| 키 | 동작 |
|---|---|
| `←` / `a` / `h` | 왼쪽 이미지 선택 |
| `→` / `d` / `l` | 오른쪽 이미지 선택 |
| `Ctrl+Z` / `Cmd+Z` | 마지막 선택 되돌리기 (Undo) |
| 좌/우 클릭 | 직접 클릭하여 선택 |

### 토너먼트 특징

- 각 라운드의 진행 상황(현재 매치 / 전체 매치 수)이 표시됩니다.
- 홀수 개의 후보가 있으면 마지막 후보는 다음 라운드로 자동 통과(Bye round)합니다.
- 후보들은 매 라운드 시작 시 무작위 순서로 재배치됩니다.
- 마지막 한 장이 자동으로 "최종 우승"으로 표시됩니다.
- 우승 이미지를 "이 이미지 선택 완료"로 확정하면 대표로 지정됩니다.
- 도중에 종료해도 그때까지의 평가는 반영됩니다.
- `Ctrl+Z`로 이전 선택을 되돌릴 수 있습니다.

## 5.7 선택 모드 (일괄 작업)

갤러리 뷰 또는 테이블 뷰에서 여러 조합을 한 번에 선택하여 일괄 작업을 수행할 수 있습니다.

### 선택 모드 진입 방법

- **롱 프레스** — 이미지/행 위에서 500ms 이상 누르면 선택 모드가 활성화됩니다.
- **우클릭 컨텍스트 메뉴** — "선택하기" 클릭.
- **Ctrl+클릭** / **Cmd+클릭** — 개별 토글.
- **Shift+클릭** — 마지막 선택 항목부터 현재 항목까지 범위 선택.

### 선택 모드에서 할 수 있는 일

선택 모드 바(도구막대 아래)에 다음 버튼이 표시됩니다.

- **재생성** — 선택된 모든 조합을 일괄 재생성 (재생성 설정 대화상자에서 개수, 템플릿, 워크플로우 선택)
- **다운로드** — 선택된 조합의 대표 이미지를 ZIP으로 다운로드
- **X** — 선택 모드 종료

선택 모드 중 `Esc` 키로도 종료할 수 있습니다.

### 컨텍스트 메뉴 (우클릭)

갤러리/테이블 뷰에서 우클릭하면 다음 메뉴가 표시됩니다.

- **열기** — 해당 조합의 상세 뷰로 이동
- **선택하기 / 선택 해제** — 선택 모드 토글
- **재생성** — 개별 조합 재생성 (파일명별 자유 그룹화 모드에서만)

## 5.8 미배정 항목 패널

축을 바꾸면 가끔 어느 그룹에도 속하지 않는 이미지가 생깁니다. 설정 드롭다운에서 **미할당** 항목을 클릭하여 패널을 엽니다.

### 미배정 패널 기능

- **미할당 이미지 목록** — 현재 축 기준으로 분류되지 않은 모든 이미지가 표시됩니다.
- **템플릿 소속 확인** — 모든 저장된 템플릿에 대해 해당 이미지가 속하는지 확인합니다. 확인 후 각 이미지마다 소속 템플릿이 표시됩니다.
- **완전 고아(소속 없음)만 보기** — 어떤 템플릿에도 속하지 않는 "완전 고아" 이미지만 필터링합니다.
- **전체 선택/해제** — 모든 미할당 이미지를 한 번에 선택하거나 해제합니다.
- **선택 항목 휴지통으로** — 선택된 미할당 이미지를 일괄 휴지통으로 이동합니다.

미배정 패널은 메타데이터가 빠져 있는 옛 이미지나 템플릿 변경 후 고아화된 이미지를 정리할 때 유용합니다.

## 5.9 필터 시스템

도구막대의 **필터** 버튼을 누르면 접이식 필터 바가 펼쳐집니다.

### 상태 필터

| 옵션 | 설명 |
|---|---|
| 전체 상태 | 모든 조합 표시 |
| 완료만 | 대표 이미지가 승인된 조합만 표시 |
| 미완료만 | 아직 대표가 지정되지 않은 조합만 표시 |

### 통합 검색

검색바에 키워드를 입력하거나 태그를 추가하여 조합을 필터링합니다. 자동완성 후보가 표시됩니다.

| 접두사 | 검색 범위 | 예시 |
|---|---|---|
| `@` | 파일명 매칭 | `@portrait` → 파일명에 `portrait`가 포함된 조합 |
| `$` | 메타데이터 매칭 | `$anime` → 메타데이터에 `anime`이 포함된 조합 |
| 없음 | 파일명 또는 메타데이터 모두 | `cat` → 파일명 또는 메타데이터 중 하나에 `cat`이 포함된 조합 |

- 여러 태그는 **AND 방식**으로 결합됩니다 (모든 조건을 만족해야 함).
- 입력 중인 텍스트로도 실시간 필터링됩니다.
- **초기화** 버튼으로 모든 필터를 한 번에 해제할 수 있습니다.
- 필터 결과 개수가 표시됩니다 (예: `42 / 120`).

## 5.10 재생성 대화상자

상세 뷰에서 `R` 키 또는 재생성 버튼을 누르면 재생성 설정 대화상자가 열립니다.

### 설정 항목

| 항목 | 설명 |
|---|---|
| **생성 갯수** | 1~64 사이의 개수 입력 (기본값: 4) |
| **ComfyUI 워크플로우 선택** | 저장된 워크플로우 또는 과거 사용 내역(History)에서 워크플로우 선택. 선택 시 미리보기 표시. |
| **사용할 템플릿** | 현재 편집 중인 템플릿, 저장된 템플릿 프리셋, 과거 사용 내역에서 선택. 선택 시 템플릿 미리보기 표시. |

재생성은 작업 큐 매니저에 새 작업으로 등록됩니다. 시드는 `random` 모드로 자동 생성됩니다.

### 일괄 재생성

선택 모드에서 여러 조합을 선택 후 **재생성** 버튼을 누르면 선택된 모든 조합이 일괄 재생성됩니다. 각 조합마다 지정된 개수만큼 새 후보가 추가됩니다.

## 5.11 일괄 동작

도구막대 우측과 선택 모드 바에는 다음과 같은 일괄 도구가 있습니다.

- **내보내기** — 승인된 대표 이미지들을 데이터셋으로 내보냅니다. 중복 파일명 구분 전략(HASH/NUM)이 적용됩니다. 진행 중에는 로딩 표시가 표시됩니다.
- **일괄 다운로드** — 선택된 조합의 대표 이미지들을 ZIP으로 다운로드합니다. 탈락/휴지통 이미지는 제외됩니다.
- **일괄 재생성** — 선택된 조합을 일괄 재생성합니다.

각 동작은 진행 결과 메시지가 도구막대 아래에 표시됩니다.

## 5.12 진행률 표시

도구막대에 진행률 바와 퍼센트가 표시됩니다.

- **진행률 바** — 대표 이미지가 승인된 조합 수 / 전체 조합 수
- **퍼센트** — 완료 퍼센트와 개수 (예: `65% (78/120)`)

## 5.13 도구막대 설정

설정 드롭다운(톱니바퀴 아이콘)에서 다음 옵션을 토글합니다.

| 설정 | 설명 |
|---|---|
| **새로고침** | 이미지 데이터를 다시 불러옵니다. |
| **리젝 숨기기** | 상세 뷰에서 탈락된 이미지를 숨깁니다. |
| **자동 다음 이동** | 대표 선택 후 다음 미완료 조합으로 자동 이동합니다. |
| **중복 전략** | 내보내기 시 중복 파일명 처리 방식: `HASH`(해시값 추가) 또는 `NUM`(번호 추가) |
| **미할당** | 미할당 이미지 패널 열기 (미할당 파일 수와 이미지 수 표시) |

## 5.14 썸네일 크기 조절

도구막대의 슬라이더로 썸네일 크기를 120px~320px까지 조절할 수 있습니다. `Ctrl+휠`로도 조절 가능합니다.

- 갤러리 뷰, 그리드 뷰, 토너먼트 뷰에서 적용됩니다.
- 크기 설정은 브라우저에 저장되어 다음 세션에도 유지됩니다.

## 5.15 모바일 지원

큐레이션 탭은 모바일에서도 완전히 작동합니다.

- **모바일 사이드바** — 상세 뷰에서 조합 목록 버튼(목록 아이콘)을 누르면 왼쪽에서 시트가 슬라이드하여 조합 목록을 표시합니다.
- **모바일 필터** — 필터 버튼을 누르면 하단에서 시트가 올라와 필터, 설정, 썸네일 크기 조절 등 모든 옵션을 제공합니다.
- **스크롤 상단 이동** — 페이지를 아래로 스크롤하면 우하단에 플로팅 버튼이 나타나 한 번에 상단으로 이동합니다.
- **반응형 레이아웃** — 상세 헤더가 모바일에서는 2단, 데스크톱에서는 1단으로 표시됩니다.

## 5.16 작업 흐름 예시

대형 데이터셋을 만드는 일반적인 흐름입니다.

1. 작업 탭에서 CEG 템플릿을 작성하고 **배치 수**를 3~5로 두어 같은 조합을 여러 번 돌립니다.
2. 작업이 모두 끝나면 큐레이션 탭을 엽니다.
3. 축을 **현재 편집 중인 템플릿**으로 두면 조합별로 후보가 묶입니다.
4. **자동 다음 이동**을 켜고, 그룹을 클릭해 상세 뷰로 들어갑니다.
5. `1~9` 키로 마음에 드는 한 장을 지정합니다. 자동으로 다음 미완료 조합으로 넘어갑니다.
6. 후보가 많으면 **토너먼트 뷰**로 전환하여 두 장씩 비교하며 선택합니다.
7. 후보가 다 별로면 `R`으로 추가 생성을 요청합니다. 작업 큐 매니저에 새 작업이 들어갑니다.
8. **필터**로 미완료 조합만 남기고 확인합니다.
9. 모두 끝나면 **내보내기**로 데이터셋 ZIP을 받습니다.
10. 정리 단계에서 **미할당 패널**을 열어 고아 이미지를 확인하고 필요시 휴지통으로 처리합니다.

## 5.17 단축키 (큐레이션)

### 상세 뷰 (그리드 모드)

| 키 | 동작 |
|---|---|
| `↓` / `j` | 다음 조합 |
| `↑` / `k` | 이전 조합 |
| `1` ~ `9` | 해당 인덱스 후보를 대표로 승인 |
| `←` / `h` | 왼쪽 이미지 포커스 |
| `→` / `l` | 오른쪽 이미지 포커스 |
| `Enter` / `Space` | 현재 포커스 이미지 선택 |
| `r` / `R` | 조합 이미지 재생성 |
| `Esc` | 조합 목록으로 복귀 |

### 토너먼트 뷰

| 키 | 동작 |
|---|---|
| `←` / `a` / `h` | 왼쪽 이미지 선택 |
| `→` / `d` / `l` | 오른쪽 이미지 선택 |
| `Ctrl+Z` / `Cmd+Z` | 되돌리기 (Undo) |

### 선택 모드

| 키 | 동작 |
|---|---|
| `Esc` | 선택 모드 종료 |

전체 단축키는 화면 어디서나 `?` 키로 띄울 수 있습니다.
