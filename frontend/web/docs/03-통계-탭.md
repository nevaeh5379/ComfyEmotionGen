# 3. 통계 탭

통계 탭은 현재 세션의 작업 진행 상황과 워커 사용량을 한눈에 보여 줍니다. 진행 중인 큰 배치를 모니터링하거나, 끝난 뒤 회고할 때 사용합니다.

표시되는 모든 수치는 **헤더에서 선택한 세션**의 작업만 집계합니다. 전체 작업을 보려면 세션 선택기에서 "전체" 또는 다른 세션으로 바꾸세요.

## 3.1 요약 카드

화면 맨 위에 핵심 지표가 가로로 늘어선 카드 행이 있습니다.

| 카드 | 의미 |
|---|---|
| 총 작업 | 세션에 등록된 모든 작업 수입니다 |
| 완료 | 정상적으로 끝난 작업 수입니다 |
| 실행중 | 지금 워커에서 처리 중인 작업 수입니다 |
| 대기 / 큐잉 | 아직 워커에 내려가지 않은 대기 작업 수입니다 |
| 실패 | 오류로 끝난 작업 수입니다 |
| 취소 | 사용자가 취소한 작업 수입니다 |

값이 변할 때 자연스럽게 강조 애니메이션이 나타나서 큰 변화가 있을 때 눈에 잘 띕니다. 값이 0인 카드는 옅게 표시되어 시각적으로 가려집니다.

## 3.2 상태 분포

원형 차트(파이) 위젯이 각 상태가 차지하는 비율을 보여 줍니다.

- 색상은 카드와 동일한 상태 색을 따릅니다 (대기·큐잉은 파랑 계열, 실행은 청록, 완료는 초록, 실패는 빨강, 취소는 회색).
- 차트 위에 마우스를 올리면 정확한 개수와 비율 툴팁이 나옵니다.

## 3.3 시간대별 처리 그래프

세션의 작업이 시간에 따라 어떻게 처리됐는지를 막대그래프로 보여 줍니다.

- 가로축은 시각, 세로축은 그 시간 구간에 들어간 작업 수입니다.
- 상태별로 색이 누적되어, 같은 시각에 완료·실패·취소가 얼마나 섞여 있었는지 한 번에 볼 수 있습니다.
- 큰 배치를 돌릴 때 워커 처리량의 안정성을 확인하는 데 유용합니다.

## 3.4 워커 사용량

등록된 ComfyUI 워커별로 다음을 표시합니다.

- 현재 상태(가용 / 점유 / 오프라인)
- 처리 중인 작업
- 누적 처리 수와 평균 처리 시간

워커가 자주 오프라인으로 빠진다면 백엔드 로그를 확인하세요. 워커 추가·삭제는 [설정 탭](./07-설정-및-단축키.md#워커-관리)에서 합니다.

## 3.5 자주 묻는 것

**Q. 통계가 갑자기 적게 잡혀요.**
A. 세션이 새로 만들어지면 새 세션부터 다시 카운트됩니다. 헤더의 세션 선택기에서 이전 세션을 선택하면 그 세션의 통계가 보입니다.

**Q. 실행 중 작업이 0인데 큐가 줄지 않습니다.**
A. 큐가 일시중지된 상태일 수 있습니다. 헤더 오른쪽의 일시중지 버튼을 확인하세요. 워커가 모두 오프라인이어도 같은 증상이 납니다.

**Q. 차트가 비어 있어요.**
A. 새 세션이거나, 시간 구간에 들어간 작업이 없을 때 그렇습니다. 작업이 한 건이라도 끝나면 막대가 채워집니다.
