# 1. 시작하기

이 장에서는 ComfyEmotionGen 웹의 전반적인 화면 구성과 처음 사용할 때 알아둘 점을 설명합니다.

## 화면 구성

화면은 크게 두 영역으로 나뉩니다.

- **헤더(상단)** — 로고, 메인 탭, 백엔드/워커 상태, 세션 선택기, 실행 컨트롤, 일시중지·취소 컨트롤이 있습니다.
- **본문(하단)** — 현재 선택한 탭의 내용이 표시됩니다.

화면이 좁아지면 헤더의 일부 컨트롤이 자동으로 축소되어 아이콘만 남거나 더보기 메뉴 안으로 들어갑니다. 좁은 모바일 화면에서는 좌측 햄버거 메뉴에 모든 항목이 모입니다.

## 메인 탭

헤더 가운데에 있는 6개 탭으로 화면을 전환합니다.

| 탭 | 단축키 |
|---|---|
| 작업 | `Alt+1` 또는 `Ctrl+Shift+1` |
| 통계 | `Alt+2` 또는 `Ctrl+Shift+2` |
| 갤러리 | `Alt+3` 또는 `Ctrl+Shift+3` |
| 큐레이션 | `Alt+4` 또는 `Ctrl+Shift+4` |
| 설정 | `Alt+5` 또는 `Ctrl+Shift+5` |

> 텍스트 입력 박스에 포커스가 있을 때는 단축키가 작동하지 않습니다.

## 상태 표시기

헤더 우측에는 두 개의 상태 표시기가 있습니다.

- **백엔드 상태등** — 백엔드 서버와의 연결 상태입니다. 초록색이면 정상이고, 회색이면 연결되지 않은 상태입니다. 잠시 끊겨도 일정 주기로 다시 시도합니다.
- **작업자 상태등** — 등록된 ComfyUI 워커의 가용 상태입니다. 클릭하면 워커 목록과 큐 현황을 볼 수 있습니다.

연결이 끊긴 상태에서도 작업 구성과 갤러리 탐색은 가능하지만, "실행" 버튼은 비활성화됩니다.

## 백엔드 주소 설정

기본 백엔드 주소는 자동으로 들어가 있지만, 다른 서버에 연결하려면 **설정 → 백엔드** 항목에서 변경합니다.

- 패키지 모드(설치형)에서는 백엔드 주소가 고정되어 변경할 수 없습니다.
- 변경 직후 백엔드 상태등이 초록색으로 바뀌면 정상입니다.

## 세션

세션은 "한 번의 작업 묶음"을 의미합니다. 헤더 가운데의 세션 선택기에서 현재 세션을 바꾸거나 새 세션을 만들 수 있습니다.

- 작업을 새로 시작할 때 **새 세션**을 만들면 작업 큐가 분리되어 깔끔하게 관리됩니다.
- 작업 큐 매니저와 통계는 선택한 세션 단위로 계산됩니다.
- 비어 있는 세션은 자동으로 정리됩니다.

세션을 사용하는 자세한 흐름은 [작업 탭](./02-작업-탭.md#세션-관리) 문서를 참고하세요.

## 패널 모드 (창 분리 / 도킹)

데스크톱에서는 `작업`·`통계`·`갤러리`·`큐레이션` 패널을 **창 모드**로 분리하거나 작업 탭 안에 **도킹**할 수 있습니다.

- 패널 헤더의 ↗ 아이콘을 누르면 별도의 플로팅 창으로 분리됩니다.
- 플로팅 창을 화면 가장자리(좌/우/상/하)로 끌어다 놓으면 작업 탭에 도킹됩니다. 가장자리에 가까이 가면 강조 영역이 나타납니다.
- 도킹된 패널은 작업 구성 패널과 함께 좌우 또는 상하로 배치됩니다.

이 기능은 **설정 → 일반 → 창 모드 사용**이 켜져 있을 때 활성화됩니다.

## 기본 작업 흐름

대부분의 사용자는 다음 흐름을 따릅니다.

1. **워크플로우 준비** — ComfyUI에서 내보낸 워크플로우 JSON을 작업 탭의 워크플로우 에디터에 붙여 넣고 저장합니다.
2. **노드 매핑** — 워크플로우의 어떤 노드 입력에 무엇을 끼울지(프롬프트, 시드, 파일명 등) 한 번만 설정합니다.
3. **CEG 템플릿 작성** — 어떤 조합으로 생성할지를 CEG DSL로 작성합니다. 기존 템플릿을 불러오거나 [템플릿 생성기](./06-템플릿-생성기.md)에서 변수만 채워 만들 수도 있습니다.
4. **미리보기 및 실행** — 예상 생성 수를 확인하고 실행 버튼을 누릅니다. 필요하면 축 필터·선택 실행·랜덤 실행을 사용해 일부만 돌립니다.
5. **결과 검토** — 갤러리에서 결과를 보거나, 같은 조합의 여러 후보 중 하나를 골라야 한다면 큐레이션 탭으로 갑니다.

다음 장부터 각 탭의 자세한 사용법을 다룹니다.
