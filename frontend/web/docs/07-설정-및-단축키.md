# 7. 설정 및 단축키

이 장은 설정 탭의 모든 항목과 화면 어디서나 쓰는 키보드 단축키를 정리합니다.

## 7.1 설정 화면 열기

헤더의 **설정** 탭 또는 단축키 `Alt+5` / `Ctrl+Shift+5`로 엽니다. 화면은 큰 섹션 단위로 나뉘어 있고, 각 섹션 안에 항목별 설명과 토글 / 입력 / 셀렉트가 들어 있습니다.

> 변경 사항은 즉시 적용되며, 별도의 "저장" 버튼은 없습니다.

## 7.2 백엔드

- **백엔드 주소** — 백엔드 서버 URL입니다. 기본은 `http://localhost:8000`. 다른 서버로 옮기거나 원격 백엔드에 연결할 때 변경합니다. 패키지 모드에서는 잠겨 있습니다.
- **헬스 체크 주기** — 백엔드 연결 상태를 확인하는 간격입니다(고급).

## 7.3 일반

- **창 모드 사용** — 데스크톱에서 작업 구성·갤러리·통계·큐레이션 패널을 별도 창으로 분리하고 가장자리 도킹할 수 있게 합니다.
- **호버 미리보기** — 갤러리와 큐레이션에서 썸네일에 마우스를 올렸을 때 큰 미리보기를 띄울지 결정합니다. 끄면 가벼워집니다.
- **승인 시 나머지 자동 탈락** — 큐레이션에서 한 후보를 승인하면 나머지 후보를 자동으로 탈락 처리합니다.
- **이미지 페이지 크기** — 갤러리 한 페이지에 표시할 이미지 수입니다. 큰 데이터셋에서는 작게 두는 게 빠릅니다.
- **이미지 지연 로딩** — 화면에 보이는 만큼만 이미지를 로딩합니다. 일반적으로 켜 두는 편이 빠릅니다.
- **단일 다운로드 모드** — 우클릭 다운로드를 ZIP 없이 개별 파일로 받습니다.

## 7.4 테마

- **테마** — 라이트 / 다크 / 시스템(OS 기본을 따름) 중 선택합니다.

## 7.5 워커 관리

ComfyUI 워커 노드를 등록·삭제합니다.

- **워커 추가** — URL과 라벨을 입력해 새 워커를 등록합니다.
- **워커 목록** — 각 워커의 상태, 현재 작업, 누적 처리량이 보입니다.
- **워커 삭제** — 휴지통 아이콘을 누르면 확인 후 삭제됩니다. 진행 중인 작업이 있으면 끝난 뒤 정리됩니다.

워커 사용량 통계는 [통계 탭](./03-통계-탭.md#34-워커-사용량)에도 표시됩니다.

## 7.6 웹훅

배치가 끝났을 때 외부 서비스(Slack, Discord, 자체 API)로 알림을 보낼 수 있습니다.

- **활성화 토글** — 웹훅 사용 여부를 켜고 끕니다.
- **엔드포인트 URL** — POST 요청을 받을 주소를 입력합니다.
- **헤더 / 페이로드 템플릿** — 필요하다면 인증 헤더와 보낼 JSON 본문 형식을 지정합니다.
- **테스트 전송** — 현재 설정으로 한 번 실제 호출을 보내 결과를 확인합니다.

## 7.7 데이터 가져오기 / 내보내기

저장된 템플릿과 워크플로우 같은 사용자 데이터를 백업할 수 있습니다.

- **내보내기** — 사용자 데이터(템플릿·워크플로우 · 노드 매핑 프리셋·설정 일부)를 JSON으로 다운로드합니다.
- **가져오기** — JSON을 업로드해서 같은 항목을 복원합니다. 이름이 충돌하면 새 이름으로 추가됩니다.

## 7.8 업데이트

웹 번들의 빌드 정보가 표시되고, 새 버전을 확인합니다.

- **현재 버전** — `BUNDLE_VERSION`과 커밋 해시가 보입니다.
- **업데이트 채널** — 안정(stable) / 베타(beta) 중 선택합니다.
- **업데이트 확인** — 채널의 최신 버전과 비교해 새 버전 안내를 표시합니다.

## 7.9 키보드 단축키 안내

화면 어디에서나 `?` 키를 누르면 단축키 안내 대화상자가 열립니다. 탭별로 다음 카테고리가 있습니다.

### 전역 단축키

| 키 | 동작 |
|---|---|
| `?` | 단축키 안내 열기 / 닫기 |
| `Alt+1..5` 또는 `Ctrl+Shift+1..5` | 메인 탭 이동 (1=작업, 2=통계, 3=갤러리, 4=큐레이션, 5=설정) |
| `Ctrl+Enter` / `Cmd+Enter` | 작업 탭에서 큐 생성 및 실행 |
| `Ctrl+S` / `Cmd+S` | 에디터 빠른 저장 |
| `Ctrl+Shift+R` / `Cmd+Shift+R` | 갤러리 목록 새로고침 |

### 갤러리 단축키

| 키 | 동작 |
|---|---|
| `←` `→` `↑` `↓` (또는 `h` `l` `k` `j`) | 이미지 포커스 이동 |
| `Enter` / `Space` | 상세 뷰어 열기 / 다중 선택 토글 |
| `1` | 이미지 통과 |
| `2` | 이미지 탈락 |
| `3` | 휴지통 이동 / 복원 |
| `p` | 비교 핀 토글 |
| `Esc` | 포커스 / 상세 뷰어 닫기 |

### 큐레이션 단축키

| 키 | 동작 |
|---|---|
| `↑` `↓` (또는 `k` `j`) | 이전 / 다음 조합 |
| `1` ~ `9` | 해당 인덱스 후보를 대표로 승인 |
| `r` / `R` | 이 조합 추가 생성 |
| `p` | 비교 핀 토글 |
| `Esc` | 조합 목록으로 복귀 |

> 텍스트 입력 칸이나 검색 박스에 포커스가 있는 동안에는 단축키가 작동하지 않습니다. 입력 칸 바깥을 클릭하거나 `Esc`로 포커스를 해제한 뒤 사용하세요.
