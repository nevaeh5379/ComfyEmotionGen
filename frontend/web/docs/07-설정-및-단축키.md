# 7. 설정 및 단축키

이 장은 설정 탭의 모든 항목과 화면 어디서나 쓰는 키보드 단축키를 정리합니다.

## 7.1 설정 화면 열기

헤더의 **설정** 탭 또는 단축키 `Alt+6` / `Ctrl+Shift+6`으로 엽니다. 화면은 큰 섹션 단위로 나뉘어 있고, 각 섹션 안에 항목별 설명과 토글 / 입력 / 셀렉트가 들어 있습니다.

> 변경 사항은 즉시 적용되며, 별도의 "저장" 버튼은 없습니다. 설정은 `appSettings` 키로 localStorage에 저장되고, 서버 WebSocket 연결을 통해 다른 탭/세션과 실시간 동기화됩니다.

## 7.2 서버 설정

### 백엔드 서버 URL

백엔드 서버 URL입니다. 기본은 `http://localhost:8000`. 다른 서버로 옮기거나 원격 백엔드에 연결할 때 변경합니다.

- **포터블(패키지) 모드**에서는 잠겨 있으며, 런처가 할당한 백엔드 포트에 자동 연결됩니다.
- 표준 설치 모드에서는 자유롭게 수정 가능합니다.

### ComfyUI 워커

여러 ComfyUI 인스턴스를 등록하면 작업이 idle(대기 중) 워커에 자동 분배됩니다.

- **워커 추가** — ComfyUI URL을 입력하고 "추가" 버튼을 누르거나 Enter를 칩니다.
- **워커 목록** — 각 워커의 ID, URL, 상태가 표시됩니다.
  - 상태 표시등: 🟢 idle(대기 중), 🟡 busy(작업 중), 🔴 down(연결 끊김)
- **워커 삭제** — 마우스를 올리면 나타나는 X 버튼을 누릅니다.
  - 진행 중인 작업이 있으면 "작업 취소 후 삭제" / "취소" 옵션이 표시됩니다.
  - "작업 취소 후 삭제"를 선택하면 해당 워커의 현재 작업을 취소하고 워커를 제거합니다.

워커 사용량 통계는 [통계 탭](./03-통계-탭.md#34-워커-사용량)에도 표시됩니다.

## 7.3 이미지 최적화

갤러리 렌더링 성능과 관련된 설정입니다.

### 갤러리 페이지당 이미지 수

갤러리 탭의 한 페이지에 표시할 이미지 수입니다. 큰 데이터셋에서는 작게 두는 게 빠릅니다.

| 옵션 | 설명 |
|---|---|
| 24장 | 데이터셋이 매우 클 때 |
| 48장 | 기본값 |
| 96장 | 작은 데이터셋 / 빠른 탐색 |

### 이미지 지연 로딩

화면 밖 이미지는 스크롤 시점에 로드합니다. 많은 이미지를 다룰 때 초기 로딩 속도가 빨라집니다.

- 기본값: **켬**

## 7.4 큐레이션 동작

큐레이션 탭과 갤러리 내보내기 동작을 제어합니다.

### 리젝 자동 적용

큐레이션에서 이미지를 선택하면 나머지를 자동으로 탈락 처리하고 다음 조합으로 이동합니다.

- 기본값: **켬**

### 빈 폴더 숨기기

큐레이션 탭에서 생성된 이미지가 없는 빈 폴더(조합)를 목록에서 숨깁니다.

- 기본값: **켬**

### 내보내기 범위

갤러리 내보내기 시 다운로드할 이미지 범위를 선택합니다.

| 옵션 | 설명 |
|---|---|
| 통과된 이미지만 | 기본값 — 승인(Approved) 상태 이미지만 내보내기 |
| 전체 이미지 | 모든 상태의 이미지를 포함 |

### 내보내기 중복 처리

동일 파일명 이미지 내보내기 시 중복 처리 방식을 선택합니다.

| 옵션 | 설명 |
|---|---|
| HASH 기반 (해시로 고유명) | 기본값 — 파일 해시를 고유 식별자로 사용 |
| NUM 기반 (숫자 순번) | 숫자 순번을 붙여 중복 방지 |

### 단일 이미지 다운로드

이미지를 클릭했을 때 다운로드 방식을 선택합니다.

| 옵션 | 설명 |
|---|---|
| 새탭에서 열기 | 기본값 — 브라우저 새 탭에서 이미지 표시 |
| 바로 다운로드 | 파일로 직접 다운로드 |

## 7.5 진행률 표시

세션 전체 진행률 표시에 적용할 계산 방식을 선택합니다.

| 옵션 | 계산식 |
|---|---|
| 완료만 / 전체 | 완료 수 ÷ 전체 수 |
| 완료 + 취소 / 전체 | (완료 + 취소) 수 ÷ 전체 수 |
| 완료 + 실패 / 전체 | (완료 + 실패) 수 ÷ 전체 수 |
| 완료 / (전체 - 실패 - 취소) | 완료 수 ÷ (전체 수 - 실패 - 취소) |

- 기본값: **완료만 / 전체**

## 7.6 UI 기능

### 호버 기능 사용

이미지 위에 마우스를 올릴 때 확대 보기와 미리보기 카드를 표시합니다. 성능이 낮은 환경에서 끄면 유용합니다.

- 기본값: **끔**

### 창 모드 기능 사용

패널을 창 모드로 분리하거나 마우스 드래그로 외부 창으로 분리할 수 있습니다. 데스크톱 환경에서 작업 구성, 갤러리, 통계, 큐레이션 패널을 별도 창으로 분리하고 가장자리 도킹할 수 있게 합니다.

- 기본값: **끔**

### 그리드 화면 맞춤(유동형) 사용

그리드의 빈 공간을 가득 채우도록 이미지 카드를 가로로 확장합니다. 비활성화 시 설정된 썸네일 고정 크기대로 렌더링됩니다.

- 기본값: **켬**

## 7.7 알림 설정 (웹훅)

배치 완료나 작업 실패 시 외부 서비스로 알림을 보낼 수 있습니다. **여러 개의 웹훅을 동시에 등록**할 수 있으며, 각 웹훅별로 채널 타입과 알림 이벤트를 독립적으로 설정합니다.

### 지원 채널 타입

| 채널 | 설명 | URL 예시 |
|---|---|---|
| Discord | Discord 웹훅 | `https://discord.com/api/webhooks/...` |
| Telegram | Telegram 봇 | Bot TOKEN 또는 전체 API URL |
| Generic | 일반 HTTP POST | `https://example.com/webhook` |

### 알림 이벤트

각 웹훅마다 다음 이벤트를 개별 켜고 끌 수 있습니다.

| 이벤트 | 설명 |
|---|---|
| 잡 완료 (`job_done`) | 개별 작업이 성공적으로 완료되었을 때 |
| 잡 실패 (`job_error`) | 개별 작업이 실패했을 때 |
| 배치 완료 (`batch_completed`) | 전체 배치 처리가 완료되었을 때 |

### 결과 이미지 포함

알림에 결과 이미지를 포함할지 선택할 수 있습니다. 이미지가 포함되면 알림 페이로드에 이미지 데이터가 추가됩니다.

### 웹훅 관리

- **웹훅 추가** — 이름, 채널 타입, URL, 이벤트 선택 후 추가
- **활성화/비활성화** — 벨 아이콘으로 개별 토글
- **테스트 전송** — 현재 설정으로 실제 알림을 한 번 전송하여 연결 확인
- **편집** — 눈 아이콘으로 이벤트 및 이미지 포함 설정 수정
- **삭제** — 휴지통 아이콘으로 삭제 (확인 창 표시)
- **URL 가리기** — 보안상 URL의 마지막 부분이 마스킹되어 표시되며, 눈 아이콘으로 토글

## 7.8 데이터 관리

저장된 템플릿, 워크플로우, 데이터베이스를 백업하거나 복원할 수 있습니다.

### ComfyUI 템플릿

저장된 ComfyUI 프롬프트 템플릿 목록을 JSON 파일로 백업하거나 가져와 현재 목록에 병합합니다.

- **내보내기** — `templates_YYYY-MM-DD.json` 파일로 다운로드
- **가져오기** — JSON 파일 업로드, 현재 목록에 병합 (이름 충돌 시 새 ID 부여)
- 저장된 템플릿 개수가 표시됩니다.

### ComfyUI 워크플로우

저장된 ComfyUI 워크플로우 및 노드 매핑 프리셋 목록을 JSON 파일로 백업하거나 가져옵니다.

- **내보내기** — `workflows_YYYY-MM-DD.json` 파일로 다운로드 (워크플로우 + 매핑 프리셋 포함)
- **가져오기** — JSON 파일 업로드, 현재 목록에 병합
- 저장된 워크플로우 개수가 표시됩니다.

### 생성 작업 데이터베이스 (SQLite)

서버의 SQLite 생성 이력 및 작업 데이터베이스(`jobs.db`) 파일 자체를 백업하거나 복원하여 덮어씁니다.

- **백업** — 서버의 `jobs.db` 파일을 다운로드
- **복원** — `jobs.db` 파일을 업로드하여 서버 데이터베이스를 덮어씀 (UI 자동 동기화)
- 서버 API(`POST /db/import`)를 통해 직접 복원되며, 진행 중 토스트 메시지가 표시됩니다.

## 7.9 업데이트

웹 번들의 빌드 정보가 표시되고, 새 버전을 확인합니다.

### 현재 버전

- 로컬 개발 환경: 커밋 해시가 표시됩니다.
- 릴리스 빌드: `BUNDLE_VERSION`과 커밋 해시가 표시됩니다.

### 업데이트 채널

새 버전을 확인할 채널입니다.

| 옵션 | 설명 |
|---|---|
| 자동 감지 | 기본값 — 빌드 채널을 자동 감지 |
| Stable | 검증된 릴리즈 버전만 확인 |
| Beta | 베타 릴리즈 포함 |
| Dev | 가장 최신 개발 빌드 |

> 로컬 개발 환경에서는 자동 업데이트가 비활성화됩니다.

새 버전이 있으면 헤더에 `↑ 태그명` 링크가 표시되어 릴리스 페이지로 이동할 수 있습니다.

## 7.10 설정 저장 및 동기화

설정은 다음과 같은 메커니즘으로 저장 및 동기화됩니다.

- **localStorage** — `appSettings` 키에 JSON으로 저장
- **서버 동기화** — WebSocket 연결을 통해 서버에 비동기 저장
- **크로스-탭 동기화** — `storage` 이벤트와 커스텀 이벤트(`__ceg_settings_ready`, `__ceg_settings_updated`)를 통해 브라우저 내 여러 탭 간 실시간 동기화
- **서버 재연결 시** — 전체 설정을 한 번에 다시 로드

### 사용 중인 localStorage 키

| 키 | 내용 |
|---|---|
| `appSettings` | 앱 환경 설정 |
| `backendUrl` | 백엔드 서버 URL |
| `theme` | 테마 설정 |
| `savedTemplates` | 저장된 프롬프트 템플릿 |
| `savedWorkflows` | 저장된 워크플로우 |
| `nodeMappings` | 노드 매핑 |
| `activeNodeMappingPresetId` | 활성 노드 매핑 프리셋 |
| `workflow` | 워크플로우 |
| `activeWorkflowId` | 활성 워크플로우 ID |
| `ceg_sessions` | 세션 데이터 |
| `ceg_active_state` | 활성 상태 |
| `cegTemplate` | 템플릿 |
| `activeTemplateId` | 활성 템플릿 ID |
| `comfy.curation.selectedAxis` | 큐레이션 선택 축 |
| `comfy.curation.thumbnailSize` | 큐레이션 썸네일 크기 |
| `comfy.gallery.thumbnailSize` | 갤러리 썸네일 크기 |

## 7.11 키보드 단축키 안내

화면 어디에서나 `?` 키를 누르면 단축키 안내 대화상자가 열립니다. 탭별로 다음 카테고리가 있습니다.

### 전역 단축키

| 키 | 동작 |
|---|---|
| `?` | 단축키 안내 열기 / 닫기 |
| `Alt+1..6` 또는 `Ctrl+Shift+1..6` | 메인 탭 이동 (1=작업, 2=통계, 3=갤러리, 4=큐레이션, 5=템플릿 생성기, 6=설정) |
| `Ctrl+Enter` / `Cmd+Enter` | 작업 탭에서 큐 생성 및 실행 |
| `Ctrl+S` / `Cmd+S` | 작업 탭(에디터)에서 빠른 저장 |
| `Ctrl+Shift+R` / `Cmd+Shift+R` | 갤러리 탭에서 목록 새로고침 |

> 탭 이동 시 토스트 알림으로 현재 탭 이름이 표시됩니다.

### 갤러리 단축키

| 키 | 동작 |
|---|---|
| `←` `→` `↑` `↓` (또는 `h` `l` `k` `j`) | 이미지 포커스 이동 (그리드 / 비교 모드) |
| `Enter` / `Space` | 상세 뷰어 열기 / 다중 선택 모드에서 선택 토글 |
| `1` | 이미지 통과 (Approved) |
| `2` | 이미지 탈락 (Rejected) |
| `3` | 휴지통 이동 / 복원 |
| `p` | 비교 핀 토글 |
| `Esc` | 포커스 초기화 / 상세 뷰어 닫기 |

### 큐레이션 단축키

| 키 | 동작 |
|---|---|
| `↑` `↓` (또는 `k` `j`) | 이전 / 다음 조합 이동 (조합 상세 뷰 전용) |
| `1` ~ `9` | 해당 인덱스 후보를 대표로 승인 |
| `r` / `R` | 현재 조합 조건에 맞추어 추가 생성 요청 |
| `p` | 비교 핀 토글 |
| `Esc` | 조합 목록으로 돌아가기 |

> 텍스트 입력 칸이나 검색 박스에 포커스가 있는 동안에는 단축키가 작동하지 않습니다. 입력 칸 바깥을 클릭하거나 `Esc`로 포커스를 해제한 뒤 사용하세요.
