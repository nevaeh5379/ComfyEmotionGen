# 4. 갤러리 탭

갤러리 탭은 지금까지 생성하고 저장한 모든 이미지를 탐색·정리하는 공간입니다. 검색, 태그, 폴더 분류, 큐레이션 상태(통과/탈락/휴지통), 비교 뷰, 데이터셋 익스포트까지 한 곳에서 처리합니다.

## 4.1 화면 구성

상단에 **갤러리 도구막대**가 있고, 그 아래로 이미지 그리드가 펼쳐집니다. 도구막대는 화면이 좁아지면 자동으로 축소되어 가장 자주 쓰는 것만 보이게 합니다.

- **검색 입력** — 파일명·태그·메타데이터 검색 칸이 들어 있습니다.
- **상태 필터** — 통과 / 탈락 / 휴지통 / 전체 등 큐레이션 상태로 좁힙니다.
- **뷰 모드** — 그리드 / 그룹 / 비교 모드 전환입니다.
- **정렬** — 시간 / 파일명 / 평점 등 정렬 기준을 선택합니다.
- **더보기 메뉴** — 익스포트, 휴지통 비우기, 그룹 재계산 등이 들어 있습니다.

## 4.2 이미지 둘러보기

### 그리드 뷰

기본 뷰입니다. 썸네일이 격자로 나열됩니다.

- 클릭: 이미지가 큰 화면(상세 뷰어)으로 열립니다.
- 마우스 위에 올림: 설정에서 "호버 미리보기"가 켜져 있으면 큰 미리보기가 즉시 표시됩니다.
- 우클릭: 컨텍스트 메뉴로 다운로드 / 복사 / 잘라내기 / 큐레이션 상태 변경 / 휴지통 / 비교 핀 등을 할 수 있습니다.

### 그룹 뷰

같은 파일명 베이스나 메타데이터 키로 묶어 봅니다. 같은 조합의 여러 시드 결과를 한 묶음으로 보고 싶을 때 유용합니다.

### 비교 뷰

여러 이미지를 한 화면에 나란히 띄워 차이점을 비교합니다.

- 그리드에서 이미지에 마우스를 올리고 `p` 키를 누르거나, 우클릭 → **비교 핀 추가**로 비교 목록에 넣습니다.
- 도구막대의 뷰 모드를 **비교**로 바꾸면 핀된 이미지들이 동시에 표시됩니다.
- 비교 뷰에서 다시 `p`를 누르면 핀이 해제됩니다.

## 4.3 검색 / 필터

도구막대의 검색 입력은 세 가지 입력 모드를 가집니다.

1. **파일명** — 부분 일치 검색입니다. 와일드카드는 없습니다.
2. **태그** — 입력 시 추천 후보가 뜹니다. 여러 태그를 OR 또는 AND로 조합합니다.
3. **메타데이터** — `key=value` 형식으로 입력합니다. 예: `emotion=happy`, `seed=12345`.

검색은 즉시 그리드에 반영됩니다. 검색 칸 옆의 ✕ 아이콘으로 한 번에 비웁니다.

### 상태 필터

도구막대의 상태 셀렉트에서 다음 중 하나를 선택할 수 있습니다.

- **대기(pending)** — 아직 분류하지 않은 이미지
- **통과(approved)** — `1` 키로 통과 처리한 이미지
- **탈락(rejected)** — `2` 키로 탈락 처리한 이미지
- **휴지통(trashed)** — `3` 키로 휴지통에 보낸 이미지
- **전체** — 모든 상태를 함께 봅니다

### 리젝 숨기기

탈락 상태를 잠시 가리고 싶다면 도구막대의 **리젝 숨기기** 토글을 켭니다. 큐레이션 작업 중에 통과 후보만 보고 싶을 때 유용합니다.

## 4.4 큐레이션 상태 바꾸기

이미지를 빠르게 분류하려면 키보드 단축키를 사용합니다.

| 키 | 동작 |
|---|---|
| `1` | 포커스된 이미지를 **통과**로 표시 |
| `2` | 포커스된 이미지를 **탈락**으로 표시 |
| `3` | 휴지통으로 이동 (휴지통 안에서는 복원) |
| `Enter` / `Space` | 상세 뷰어 열기 (다중 선택 모드에서는 선택 토글) |
| `←` / `→` / `↑` / `↓` 또는 `h j k l` | 포커스 이동 |
| `Esc` | 포커스 / 상세 뷰어 닫기 |
| `p` | 비교 핀 토글 |

여러 장을 한꺼번에 처리하려면 다중 선택 모드를 사용합니다.

1. 도구막대의 **선택 모드** 버튼을 켭니다.
2. 원하는 이미지를 클릭해 선택합니다(Shift 클릭으로 범위 선택).
3. 선택한 항목에 대해 일괄 휴지통 / 통과 / 탈락 / 다운로드 / 폴더 이동 등을 적용합니다.

## 4.5 폴더 정리

이미지를 폴더로 묶어 정리할 수 있습니다.

- 그리드 상단의 **브레드크럼**으로 현재 위치를 확인하고, 상위 폴더로 이동합니다.
- 우클릭 메뉴의 **폴더로 이동 / 복사** 항목으로 이미지를 폴더로 옮깁니다.
- **새 폴더 만들기** 항목으로 폴더를 생성합니다.

폴더 구조는 백엔드의 이미지 저장소 폴더 구조와 일치합니다.

## 4.6 노트와 태그 편집

상세 뷰어 또는 그리드 카드의 작은 편집 영역에서 다음을 즉시 입력할 수 있습니다.

- **태그** — 콤마로 구분해 여러 개 추가합니다. 자주 쓰는 태그는 자동 완성됩니다.
- **노트** — 자유 형식의 메모입니다.

저장 버튼은 없고 입력 종료 시 자동으로 반영됩니다.

## 4.7 휴지통

휴지통은 "지울지 한 번 더 생각하는" 안전망입니다.

- `3` 키나 우클릭 → 휴지통으로 이미지를 보냅니다.
- 상태 필터를 **휴지통**으로 바꾸면 휴지통 안의 이미지가 보입니다.
- 휴지통 안에서 `3`을 누르면 다시 대기(pending) 상태로 복원됩니다.
- 도구막대의 **휴지통 비우기**(더보기 메뉴 안)로 영구 삭제합니다. 확인 대화상자가 한 번 뜹니다.

## 4.8 데이터셋 익스포트

훈련용 데이터셋을 만들 때 사용합니다. 도구막대의 더보기 메뉴 → **익스포트**를 누르세요.

- **현재 필터 적용 상태**의 이미지만 묶여 ZIP으로 다운로드됩니다.
- 파일명 중복 처리는 두 가지 중 선택할 수 있습니다.
  - **해시 추가** — 중복일 때 파일명 뒤에 해시를 붙여 모두 보존합니다.
  - **번호 추가** — `_001`, `_002` 형식의 일련번호를 붙입니다.
- 이미지와 함께 메타데이터(태그·노트·축 값)가 같이 들어갑니다.

용량이 크면 시간이 걸릴 수 있습니다. 진행률 토스트가 떠 있습니다.

## 4.9 그 외 도구

- **새로고침** — `Ctrl+Shift+R` 또는 도구막대 새로고침 버튼으로 백엔드에서 다시 목록을 가져옵니다.
- **filename 그룹 재계산** — 더보기 메뉴 안. 파일명 기반 그룹이 어긋났을 때 다시 계산합니다.

## 4.10 단축키 (갤러리)

| 키 | 동작 |
|---|---|
| `←` / `h` | 이전 이미지 |
| `→` / `l` | 다음 이미지 |
| `↑` / `k` | 위쪽 이미지 |
| `↓` / `j` | 아래쪽 이미지 |
| `Enter` / `Space` | 상세 뷰어 열기 / 선택 토글 |
| `1` | 통과 |
| `2` | 탈락 |
| `3` | 휴지통 이동 / 복원 |
| `p` | 비교 핀 토글 |
| `Esc` | 포커스 해제 / 상세 뷰어 닫기 |
| `Ctrl+Shift+R` | 목록 새로고침 |

전체 단축키는 화면 어디서나 `?` 키로 띄울 수 있습니다.
